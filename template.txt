task_2:
instructions: 読みがなを書いて、その言葉の意味に合うものをア～オの中からえらびましょう。
1. 作業を兄に【（任）せる】。
2. 月が【（現）れる】。
3. 【（態度）】に出る。
4. 子犬を【（飼）う】。
5. 【（実際）】に体験する。

answers:
1. ゆだねる
2. 出てくる
3. 行動・表情
4. 本当に
5. 家で育てる

There must be an empty line above this line. All other empty lines also need to stay.
For task_1, two parentheses "（ふりがな）" create an empty text box with the text in parentheses as ruby above it. Be mindful that the parentheses must be in full-spaced （全角）.
